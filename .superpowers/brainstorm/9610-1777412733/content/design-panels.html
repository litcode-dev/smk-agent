What changes across all panels

  • Header pattern unified — every panel gets a 13px semi-bold title + muted sub-info, replacing the all-caps micro-label pattern
  • Filter tabs — segmented pill group (bg-slate-900 container, active item bg-slate-800) instead of floating buttons
  • Row left-border accent — 2px colored accent tracks status (emerald=done, sky=running, rose=failed) on agent rows and automations
  • Badge pills — rounded-full uppercase tiny pills for tier/status badges instead of square labels
  • Consolidation stepper — horizontal 5-step progress bar replaces vertical card stack; active step glows
  • Connections — emerald border + subtle tint on connected cards; dimmed opacity on disconnected
  • Consistent spacing — 8px gap-based rows, 8px border-radius on rows (up from 6px)
