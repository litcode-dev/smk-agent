Key shell changes

  • Header decluttered — memory pill counts moved to the Memory panel where they belong. Header shows only: logo, agent name, live status pill, theme toggle.
  • Live indicator — becomes a pill with a glowing emerald dot instead of a bare dot + text.
  • Sidebar active state — emerald left-border accent + very subtle tinted background instead of the current flat grey highlight.
  • Sidebar width — 168px → 196px, giving labels room to breathe and adding a divider before Connections.
  • Agent badge — stays, but uses emerald with dark green text instead of sky blue.
  • Stat cards — rounded-xl instead of rounded-lg, trend arrows with color-coded delta.
